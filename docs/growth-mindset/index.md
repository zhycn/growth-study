# 增长认知

理解增长的本质，掌握 AARRR 模型与增长黑客思维，建立数据驱动的决策框架。

## 模块概述

增长认知是增长学习的起点。在这一模块中，你将：
- 理解什么是真正的"增长"，区分增长与传统营销的差异
- 掌握 AARRR 模型（海盗指标）的核心逻辑
- 了解增长黑客的起源、演变与核心理念
- 学习增长飞轮与北极星指标的设定方法
- 探索如何建设增长型组织
