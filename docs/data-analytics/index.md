# 数据分析

指标体系、漏斗分析、A/B 测试，用数据驱动增长决策。

## 模块概述

数据分析是增长决策的基础。在这一模块中，你将：
- 学会搭建完整的指标体系
- 掌握漏斗分析、同期群分析等核心分析方法
- 深入理解 A/B 测试的设计与执行
- 学习归因分析、数据可视化与 SQL 基础
- 探索预测性分析、LTV 模型与增长建模

### 🟢 基础篇
- [指标体系搭建](/data-analytics/basics/metrics-framework) — 如何构建科学的指标体系

- [同期群分析](/data-analytics/basics/cohort-analysis) — 用户分群与留存分析

### 🟡 进阶篇
