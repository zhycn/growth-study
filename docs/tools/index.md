# 工具栈

分析工具、自动化平台、MarTech 架构，打造高效增长技术栈。

## 模块概述

工具栈是增长效率的保障。在这一模块中，你将：
- 了解主流分析工具与营销技术栈
- 深入掌握 GA4 等核心分析工具
- 学会使用自动化工具提升运营效率
- 掌握 CRM 系统选型与 CDP 客户数据平台
- 探索 MarTech 架构、AI 增长工具与自定义数据看板
